# Poketrainer

Proyecto de pruebas usando la PokeAPI.

## Estructura
- backend/: API y lógica de negocio (Python)
- frontend/: (por implementar)

## Cómo empezar
1. Clona el repositorio
2. Instala dependencias en backend:
   cd backend/src
   pip install -r requirements.txt
3. Ejecuta la API:
   python main.py
